# FOTO_BLUR

Program Python sederhana untuk memberikan efek blur pada foto secara otomatis. Program ini terinspirasi dari video-video fyp di berbagai platform media sosial yang menggunakan lagu "Foto kita blur by Sal Priadi".

## Fitur

- Blur foto secara otomatis.
- Mudah digunakan.
- Ringan dan cepat.
- Open Source.
- Dapat dimodifikasi sesuai kebutuhan.

## Requirements

- Python 3.8 atau lebih baru.
- pip.

## Instalasi

1. Download atau clone repository ini.

```bash
git clone https://github.com/USERNAME/FOTO_BLUR.git
```

Atau download file `.zip` kemudian extract.

2. Masuk ke folder proyek.

```bash
cd FOTO_BLUR
```

3. Install semua dependensi.

```bash
pip install -r requirements.txt
```

## Cara Menjalankan

Jalankan program dengan perintah berikut.

```bash
python blur.py
```

## Struktur Proyek

## Mengedit Program

Jika ingin mengubah atau mengembangkan program:

1. Buka folder proyek menggunakan Visual Studio Code (VS Code).
2. Edit file `blur.py`.
3. Simpan perubahan, lalu jalankan kembali program.
